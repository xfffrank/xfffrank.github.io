    • --name：指定容器的名字；
    • -d：detach，建立容器后，即脱离当前进程；

查看 docker 进程

  • 说明：
    • ps：present state，显示正在执行的容器。
    • 在 docker 中，一个容器对应一个进程，进程终止，容器也就释放，停止执行。

    • exec使得容器在退出后依然在后台运行，下次打开前不需要重新start
